# Freight Delivery API REST

1. Clonar Proyecto
2. Instalar adminstrador de paquetes
```
npm install
```
3. Clonar archivo ```.env.example``` y renombrarlo a ```.env```
4. Cambiar variables de entorno
5. Levantar proyecto en modo desarrollo
```
npm run start:dev
```
